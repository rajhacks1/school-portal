# School Portal - Complete Management System

A fully functional school management portal built with vanilla JavaScript and IndexedDB.

## Features

✅ **Authentication**
- User login with email/roll number and password
- User registration (Students & Teachers)
- Admin approval workflow

✅ **Admin Dashboard**
- Subject Management (Add, Edit, Delete)
- Class Management
- Teacher & Student Management
- Attendance Tracking
- Results Management
- Registration Approvals

✅ **Teacher Dashboard**
- View assigned classes
- Mark attendance
- View results

✅ **Student Dashboard**
- View attendance
- View results
- Check enrolled subjects

## Deployment

### Netlify
1. Push this folder to GitHub
2. Go to https://app.netlify.com
3. Click "New site from Git"
4. Select your repository
5. Deploy!

### Other Static Hosting
- GitHub Pages
- Vercel
- AWS S3
- Azure Static Web Apps

## Files

- `index.html` - Main HTML file
- `styles.css` - Styling
- `db.js` - Database operations
- `app.js` - Application logic
- `netlify.toml` - Netlify configuration
- `package.json` - Package metadata

## Local Development

1. Open `index.html` in your browser
2. Data is stored in IndexedDB (browser's local database)
3. Works offline after first load
